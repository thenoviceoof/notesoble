What is Arduino?

It's an easy to use microcontroller platform

Arduino is an easy way to make something smart, or bring computing power to the real world

What's a microcontroller?

A small computer, usually dedicated to one task, unlike the multi-tasking general computers that you're probably used to. For instance, smart washing machines and ??? use microcontrollers instead of a full-blown computer

Why use an Arduino?

Arduino is easy to learn

The Arduino programming language was built on Processing, a language that was meant to be picked up by non-technical people like artists. There's no need to spend days getting your first success: just upload the first demo, and you're there.

Language constructs are just enough (no OOP), wraps low-level things nicely, one-download IDE+beginning sketches, works with USB almost out of the box, don't need a dedicated programmer,

A programming language?

Arduino is (relatively) cheap
